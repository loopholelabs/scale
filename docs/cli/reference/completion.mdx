The `completion` command generates a shell script that can be used to provide shell autocompletion for the Scale CLI.
The script can be used with Bash, Zsh, Fish, or PowerShell.

## Usage

```bash
scale completion <shell>
```

Supported shells are `bash`, `zsh`, `fish`, and `powershell`.
